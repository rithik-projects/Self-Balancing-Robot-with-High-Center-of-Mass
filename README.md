# Self-Balancing-Robot-with-High-Center-of-Mass
Designed and constructed a two-wheeled self-balancing robot with a high center of mass (~0.20 m) and total mass of ~1.5 kg, using Fusion 360 and AutoCAD to develop manufacturable CAD models and fabrication drawings. Integrated a real-time control architecture consisting of an IMU (MPU6050) and encoder feedback interfaced with an ESP32, communicating via UART with DDSM210 direct-drive servo wheels through a Driver HAT. Implemented PID-based control for dynamic stabilization and, through iterative hardware testing, identified system delay and sensor latency as key sources of instability. Developed embedded C/C++ control code for sensor fusion using a complementary filter and real-time motor actuation, while acquiring feedback data such as speed, current, and temperature from the motor drivers. Transitioning to a Linear Quadratic Regulator (LQR) control framework by formulating state-space equations and tuning gain matrices to minimize latency, improve response time, and achieve robust balance under high center-of-mass conditions.
